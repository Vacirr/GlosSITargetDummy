# GlosSITargetDummy GlosSITarget.exe

A streamlined, bloat-free alternative to the original [GlosSITarget.exe](https://github.com/Alia5/GlosSI). This project removes the heavy Steam Input features of GlosSI, providing a purely performance-oriented way to capture Steam screenshots (F12) for non-Steam games.

*This project is a modified version of the original [GlosSI by Alia5](https://github.com/Alia5/GlosSI).*

## Features
* This version is completely stripped of Steam Input mechanics, meaning it consumes almost no resources and causes zero performance drops.
* It works perfectly seamlessly with the Playnite launcher.
* You can take desktop or game screenshots using the F12 key and seamlessly upload them to Steam under the game's name.

## Installation and Usage
1. First, install the original [GlosSI](https://github.com/Alia5/GlosSI) normally.
2. Replace the original `GlosSITarget.exe` file in the installation folder with this modified `GlosSITarget.exe`.
3. You can use `GlosSIconfig` just as you normally would.
4. You will still appear as playing that specific game on Steam.
5. While in-game, use `Alt+Tab` to switch to the `GlosSITarget.exe` window.
6. Press `F12` when the window is in focus.
7. A screenshot of your screen will be taken and saved directly to Steam!
8. Your status on Steam will show you playing this game until GlosSITarget is closed.
